# 🏢 大阪市24区 中古マンション投資戦略分析

## 📊 プロジェクト概要

本プロジェクトは、大阪市24区の中古マンション市場における投資機会を、データサイエンスの手法を用いて定量的に評価したものです。

**職業訓練校（5ヶ月コース）の卒業制作として作成しました。**

---

## 🎯 主要発見

### 1. 「50㎡の壁」の逆説
- **発見**: 50㎡超で平米単価が-23.7%低下
- **示唆**: 投資効率の観点から、50-70㎡が最適である可能性
- **統計的有意性**: t検定 p < 0.001

### 2. 地域差の存在
- **最高単価**: 北区 77.73万円/㎡
- **最低単価**: 平野区 47.73万円/㎡
- **価格差**: 約63%（30.00万円/㎡）

### 3. 駅近プレミアム
- **徒歩5分以内**: 68.64万円/㎡
- **徒歩6分以上**: 61.08万円/㎡
- **プレミアム**: +12.4%

## 📊 データ概要

### データ源
- 国土交通省 不動産取引価格情報
- 対象期間: 2020年Q1～2024年Q2
- 対象エリア: 大阪市24区全域

### データクレンジング
- 元データ: 18,846件
- クレンジング後: 7,138件（37.9%）
- 削除: 11,708件（62.1%）

## 💻 Streamlitアプリの使い方

### ローカルで起動

1. リポジトリをクローン
2. conda create -n osaka-env python=3.11
3. conda activate osaka-env
4. pip install streamlit plotly pandas numpy matplotlib seaborn
5. streamlit run streamlit_app.py

### アプリの機能
- 🔍 条件検索（区・面積・築年数・駅距離）
- 📊 データ可視化（インタラクティブなグラフ）
- 💰 投資シミュレーション（利回り計算）
- 📥 データダウンロード（CSVエクスポート）

## 📋 投資戦略の提案

### 🔰 初心者向け（予算: 2,000-3,000万円）
- 推奨エリア: 天王寺区、阿倍野区、福島区
- 推奨面積: 30-50㎡
- 期待利回り: 4.5-5.5%

### ⭐ 中級者向け（予算: 3,000-5,000万円）
- 推奨エリア: 中央区、西区、浪速区
- 推奨面積: 50-70㎡
- 期待利回り: 5.0-6.0%

### 💎 上級者向け（予算: 5,000万円以上）
- 推奨エリア: 北区、中央区
- 推奨面積: 70-100㎡
- 期待利回り: 4.0-5.0%
